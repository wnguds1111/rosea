ROSEA 웹 클라이언트 사전예약 기획서

기존의 위아래로 스크롤하는 웹사이트 방식을 버리고, 웹 브라우저 전체를 하나의 '라그나로크 온라인 인게임 클라이언트'처럼 작동하게 만드는 혁신적인 게이미피케이션 기획입니다. 캐릭터 생성창에서 사전예약을 완료하고, 인게임 퀘스트 창을 통해 이벤트에 참여합니다.

1
Phase 1: 접속 및 사전예약 (캐릭터 생성 UI)

일반적인 웹 랜딩 페이지가 아닌, 친숙한 [라그나로크 캐릭터 선택/생성창]이 전체 화면으로 나타납니다. 유저는 실제 게임을 시작하듯 사전예약을 진행합니다.

UI 컴포넌트기능 상세 및 연출 로직
로그인 제어 (2차 비밀번호 UI)빈 캐릭터 슬롯을 클릭하면 '2차 비밀번호 키패드' 팝업이 연출되며, 이를 통해 로그인/본인인증을 유도.
캐릭터 생성창 (사전예약 폼)닉네임 입력란이 곧 사전예약자 정보/추천인 코드 입력 필드로 작동. (헤어스타일 등 클릭하며 노는 기능 포함)
Start Game 버튼 (사전예약 완료)우측 하단의 포링 형태 [Start Game] 버튼을 누르면 인게임 진입 사운드와 함께 하얗게 화면이 트랜지션 되며 Phase 2(인게임 월드)로 진입.
사전예약 기본 보상C_10Grade_Balloon x1, 전투교범 x2, 풍선껌 x2, 노란 포션 상자 x3, Daily Spin 티켓 3장 즉시 지급
2
Phase 2: 인게임 월드 진입 (이벤트 허브 UI)

캐릭터를 생성(사전예약)하고 진입한 메인 화면입니다. 스크롤이 없는 1 화면 구조이며, 배경은 라그나로크 아카데미나 프론테라 마을로 설정되고 인게임 HUD가 배치됩니다.

UI 컴포넌트기능 상세 및 연출 로직
상태창 (좌측 상단 HP/SP 바)유저의 프로필, 레벨(가입기간 등), 그리고 보유 중인 데일리 스핀 티켓 수량을 SP 바 형태로 시각화.
시스템 채팅창 (좌측 하단)마일스톤 돌파 소식, "ㅇㅇㅇ님이 가챠에서 1등에 당첨되었습니다" 등의 서버 알림이 인게임 시스템 폰트로 실시간 타이핑됨.
단축키 메뉴 (우측 하단)인덱스 이동, 공지사항, FAQ 버튼 등을 인게임 가방/설정 아이콘 형태로 배치.
3
Phase 3: 퀘스트 창을 통한 이벤트 진행

기존 웹사이트의 스크롤 구역(섹션)을 완벽히 대체하는 [인게임 Quest 창]입니다. 유저는 화면 중앙에 뜬 퀘스트 목록을 클릭해 이벤트를 수행합니다.

퀘스트 명 (이벤트)기능 상세 및 보상 데이터
Quest 1. 마일스톤 달성내용: 글로벌 사전등록 인원 누적 게이지 표시.
보상:
- 10k: 민첩성/블레싱 10레벨 스크롤 x10
- 50k: 보험증서x3, 지크프리드x3, 파리의날개 1일 x1
- 100k: 전투교범x2, 풍선껌x2, 하얀포션상자x3
- 200k: 생명수/마나수 세트
- 300k: 10레벨 요리 세트
Quest 2. 베테랑 라운지 인증내용: 기존 GGH/GGL 플랫폼 플레이 타임 감정. 퀘스트 클릭 시 인게임 NPC 대화창 연출 등장.
보상:
- 다이아 (5년 이상): Selection Box x1 + 대형 소모품
- 플래티넘 (3년 이상): C_Mini_Glasses x1 + 중형 소모품
- 골드 (1년 이상): Lincoln_Hat x1 + 소형 소모품
- 실버 (1년 이하): 기본 포션 세트
Quest 3. 데일리 캡슐 룰렛내용: 매일 출석(95% 1장, 5% 2장) 및 초대(건당 3장)로 획득한 티켓 사용처.
연출: [퀘스트 진행] 클릭 시 퀘스트 창이 닫히고 화면 중앙에 커다란 가챠 머신 UI가 등장하여 17종 보상 룰렛 진행.

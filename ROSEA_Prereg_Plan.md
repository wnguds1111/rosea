# ROSEA 사전예약 기획서: In-Game Gamification Concept

## 핵심 컨셉: "웹사이트가 아닌, 라그나로크 접속 그 자체"
사전예약의 모든 플로우(정보 입력, 이벤트 참여, 보상 확인)를 라그나로크 온라인의 **실제 인게임 클라이언트 UI(캐릭터 생성, 퀘스트 창, 채팅창 등)**로 완벽하게 치환하여, 올드 유저들의 향수를 극대화하고 압도적인 몰입감을 제공하는 게이미피케이션 기획입니다.

---

## Phase 1: 접속 및 사전예약 (캐릭터 선택/생성)
웹사이트 최초 진입 시 마주하는 화면으로, 일반적인 스크롤 랜딩 페이지가 아닌 '라그나로크 캐릭터 선택창' 화면으로 고정 연출됩니다.

### 세부 기획
- **로그인 제어**: 빈 슬롯 클릭 시 2차 비밀번호 키패드 UI 팝업이 뜨며, 이를 통해 로그인/본인인증을 진행.
- **캐릭터 생성창 (사전예약 폼)**: `Create Character` 창이 팝업됨.
  - **닉네임 입력란**: 사전예약 시 사용할 닉네임 또는 추천인 코드 입력 필드로 활용.
  - **꾸미기 요소**: 헤어스타일, 색상을 클릭해 볼 수 있는 재미 요소(상호작용) 추가.
- **게임 시작 (사전예약 완료)**: 우측 하단의 [Start Game(포링 버튼)] 클릭 시 사전예약 데이터가 전송되며, 인게임 BGM과 함께 화면이 하얗게 페이드아웃 후 Phase 2로 진입.
- **기본 보상**: 10Grade Balloon, 전투교범x2, 풍선껌x2, 노란포션상자x3, Daily Spin 티켓 3장 즉시 지급.

---

## Phase 2: 인게임 월드 (이벤트 허브)
캐릭터 생성을 마치고 들어온 메인 화면으로, 전체 화면이 인게임 배경(아카데미/프론테라)으로 구성되며 클래식 HUD UI가 적용됩니다.

### 세부 기획
- **HUD (상태창)**: 좌측 상단 HP/SP 바에는 유저의 '데일리 티켓 보유 수량' 및 가입 기간 기반 레벨 표시.
- **채팅창 (알림창)**: 좌측 하단 시스템 채팅창에 "사전예약 누적 5만명 돌파!", "XXX님이 가챠 당첨!" 등이 실시간 텍스트로 타이핑됨.
- **단축키 창**: 우측 하단의 인게임 아이콘(가방, 설정 등)을 통해 공지사항, FAQ, 외부 링크로 이동.
- **메인 퀘스트 창 (이벤트 보드)**: 화면 중앙에 띄워진 거대한 `Quest` 창을 통해 모든 이벤트를 진행.

---

## Phase 3: 퀘스트 기반 이벤트 진행
스크롤을 내리는 대신, `Quest` 창 좌측의 목록 탭을 클릭하여 우측 상세 패널에서 해당 이벤트를 확인하고 참여합니다.

### Quest 1: 마일스톤 달성 (글로벌 사전예약)
- **상세 내용**: 퀘스트 설명란에 현재 누적 달성 인원 게이지가 프로그레스 바 형태로 노출.
- **달성 보상**:
  - 10,000명: 민첩성/블레싱 10레벨 스크롤 x10
  - 50,000명: 보험증서x3, 지크프리드x3, 파리의날개 1일 x1
  - 100,000명: 전투교범x2, 풍선껌x2, 하얀포션상자x3
  - 200,000명: 생명수/마나수 세트
  - 300,000명: 10레벨 요리 세트

### Quest 2: 베테랑 라운지 (기존 유저 인증)
- **상세 내용**: 기존 GGH/GGL 플랫폼 플레이 기록을 감정받는 퀘스트.
- **연출 플로우**: [퀘스트 진행] 클릭 시, 퀘스트창 위에 클래식 NPC 대화창이 나타나며 "당신의 모험 기록을 살펴볼까요?" 메시지 출력 후 등급 판정 연출.
- **등급별 보상**:
  - 다이아 (5년 이상): Selection Box x1 + 대형 소모품 세트
  - 플래티넘 (3년 이상): C_Mini_Glasses x1 + 중형 소모품 세트
  - 골드 (1년 이상): Lincoln_Hat x1 + 소형 소모품 세트
  - 실버 (1년 이하): 기본 포션 세트

### Quest 3: 데일리 캡슐 룰렛 (야시장 가챠)
- **상세 내용**: 매일 접속 및 친구 초대로 얻은 티켓을 소모하는 반복 일일 퀘스트.
- **티켓 획득처**: 기본 3장 / 친구초대 건당 3장(최대15장) / 일일접속 1~2장.
- **연출 플로우**: [룰렛 돌리기] 클릭 시 화면 중앙에 거대한 가챠 룰렛 기계 UI 팝업 등장. 17종의 보상 중 하나가 추첨되며, 퀘스트 완료음(띠링~) 재생.

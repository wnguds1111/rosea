# ROSEA 사전예약 기획서: Modern Web Platform Concept

## 핵심 컨셉: "직관적이고 몰입감 있는 2-Panel 퀘스트형 UI"
모던한 웹 UI 트렌드를 반영한 직관적인 사전예약 페이지 기획입니다. 전체 화면을 활용한 **2-Panel 레이아웃(좌측 비주얼 고정, 우측 콘텐츠 스크롤)**을 통해 유저에게 게임의 세계관을 지속적으로 노출하며, 사전예약부터 이벤트 참여까지의 모든 과정을 명확한 **퀘스트(Quest) 형태**로 제공하여 참여율과 유저 편의성을 극대화합니다.

---

## Q1: 사전예약 (Pre-registration)
웹사이트 최초 진입 시 우측 상단에 노출되는 가장 기본적인 퀘스트 구역입니다.
- **기능 상세**: 휴대폰 번호 또는 이메일을 통한 간편한 사전예약을 지원합니다. 약관 동의 체크박스가 포함되어 있습니다.
- **보상**: 사전예약 완료 시 인게임 아이템(10Grade Balloon, 전투교범 등) 및 Daily Machine 룰렛 티켓 +3장 즉시 지급.

---

## Q2: 마일스톤 (Milestone)
글로벌 사전예약 누적 달성 인원 수에 따라 보상이 언락되는 공용 퀘스트입니다.
- **기능 상세**: 진행률을 직관적으로 보여주는 게이지 바 형태로 시각화.
- **보상 단계**:
  - 10k: 민첩성/블레싱 10레벨 스크롤 x10
  - 50k: 보험증서x3, 지크프리드x3, 파리의날개 1일 x1
  - 100k: 전투교범x2, 풍선껌x2, 하얀포션상자x3
  - 200k: 생명수/마나수 세트
  - 300k: 10레벨 요리 세트

---

## Q3: 베테랑 라운지 (Veteran Lounge)
기존 라그나로크 온라인(RO) 유저들의 플레이 타임 데이터를 기반으로 등급을 판정하고 추가 보상을 제공하는 특별 구역입니다.
- **기능 상세**: GGH/GGL 플랫폼 계정 연동 후, 플레이 이력에 따라 다이아/플래티넘/골드/실버 등급 부여.
- **보상**:
  - 다이아 (5년 이상): Selection Box x1 + 대형 소모품
  - 플래티넘 (3년 이상): C_Mini_Glasses x1 + 중형 소모품
  - 골드 (1년 이상): Lincoln_Hat x1 + 소형 소모품
  - 실버 (1년 이하): 기본 포션 세트

---

## Q4: 데일리 머신 이벤트 (Daily Machine)
사전예약, 친구 초대, 일일 로그인을 통해 티켓을 모으고 가챠를 돌리는 메인 참여형 이벤트 구역입니다.
- **티켓 획득 룰**:
  1. 사전예약 시 +3장
  2. 친구 초대 시 건당 +3장 (최대 5명 제한)
  3. 일일 출석 체크 시 +1~2장 랜덤 (매일 UTC+8 자정 초기화)
- **가챠 머신**: 모은 티켓으로 중앙의 가챠 머신을 돌려 17종의 인게임 아이템 중 하나를 획득.
- **히스토리 관리**: 보상 획득 내역(REWARD HISTORY)과 티켓 획득 내역(TICKET HISTORY) 팝업을 분리하여 상세한 참여 이력 제공.
